JavaScript Arrays

Definition

In JavaScript, an array is a collection of elements that can hold multiple values under a single variable name. Arrays are zero-indexed and can store any data type.

Operations

  • Add Elements: `push()`
  • Remove Elements: `pop()`
  • Access Element: `at()`
  • Combine: `join()`
  • Convert to String: `toString()`
  • Get Length: `length`

Example


            // Create an array
let fruits = ["Apple", "Banana", "Cherry"];
console.log("Initial Array:", fruits);

// Add an item
fruits.push("Mango");
console.log("After push:", fruits);

// Remove an item
fruits.pop();
console.log("After pop:", fruits);

// Access an element using at()
console.log("Element at index 1:", fruits.at(1));

// Combine elements into a string using join()
let combined = fruits.join(", ");
console.log("Combined String:", combined);

// Convert the array to a string
let asString = fruits.toString();
console.log("Array as String:", asString);

// Get the length of the array
console.log("Length of Array:", fruits.length);
