JavaScript Strings

Definition

Strings in JavaScript are sequences of characters used to represent text. Strings are immutable and can be created using single, double, or backticks.

Operations

  • Concatenate: `+`, `concat()`
  • Slice: `slice()`, `substring()`
  • Length: `length`
  • Replace: `replace()`

Example


    // Create a string
    let greeting = "Hello, World!";
    console.log(greeting);
    
    // Find length
    console.log(greeting.length);
    
    // Slice a part
    console.log(greeting.slice(0, 5));
    
    // Replace text
    console.log(greeting.replace("World", "JavaScript"));
