JavaScript Operators

Definition

In JavaScript, operators are symbols or keywords used to perform operations on operands. These include arithmetic, assignment, comparison, logical, and more.

Types of Operators

  • Arithmetic Operators: +, -, *, /, %
  • Assignment Operators: =, +=, -=, *=
  • Comparison Operators: ==, ===, !=, !==, >, <
  • Logical Operators: &&, ||, !

Examples


    // Arithmetic
    let a = 10, b = 5;
    console.log(a + b); // 15
    console.log(a - b); // 5
    
    // Comparison
    console.log(a > b); // true
    console.log(a === b); // false
    
    // Logical
    console.log(a > 0 && b > 0); // true
    console.log(a > 0 || b < 0); // true
